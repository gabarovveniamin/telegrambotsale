# 🛒 Telegram Discount Monitor Bot (Kazakhstan)

Высоконагруженный Telegram-бот для автоматического мониторинга скидок на E-commerce платформах Казахстана (Technodom, Sulpak, Mechta и др.).

## 🚀 Стек технологий
- **Aiogram 3.x**: Современный асинхронный фреймворк для Telegram-ботов.
- **APScheduler**: Планировщик фоновых задач (внутрипроцессный).
- **Aiosqlite**: Асинхронное взаимодействие с базой данных SQLite.
- **curl_cffi**: Продвинутые HTTP-запросы с имитацией TLS-отпечатков браузера (Chrome) для обхода WAF/Cloudflare.

## 🛠 Архитектура
1. **Дедупликация (SQLite)**: Использование `INSERT OR IGNORE` для атомарной проверки уникальности товаров по их ID.
2. **Безопасная рассылка**: Реализована очередь рассылки с соблюдением лимитов Telegram API (~30 сообщений в секунду глобально) и обработкой `TelegramRetryAfter`.
3. **Модульность**: Код разбит на логические части: конфигурация, БД, парсер, бот и планировщик.

## 📦 Установка и запуск
1. Установите зависимости:
   ```bash
   pip install aiogram apscheduler aiosqlite curl_cffi
   ```
2. Настройте `BOT_TOKEN` в `config.py` или через переменные окружения.
3. Запустите бота:
   ```bash
   python main.py
   ```

## 📝 Особенности реализации
- **Обход блокировок**: `curl_cffi` позволяет боту выглядеть как обычный пользователь браузера Chrome, что критично для парсинга крупных маркетплейсов.
- **Производительность**: Все операции ввода-вывода (БД, Сеть, API) выполняются асинхронно, не блокируя event loop.
- **Надежность**: Планировщик интегрирован в жизненный цикл `aiogram` (startup), что гарантирует его работу только при активном боте.
